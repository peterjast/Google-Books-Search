# **Google-Books-Search**

This app searches the Google Books API and allows users to save books of interest.


## Tools:

* Javascript
* Node.js
* Express
* Heroku
* MongoDB
* Mongoose
* Axios

## How the app works

* Users can search for any book title 
* Results from the google books API are provided
* Users can save any books of interest
